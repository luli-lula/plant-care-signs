# Native Plant Cards

36 Keystone native plant cards for the Capital Region of New York.

Scan the QR code on a plant card to see its interactive page — growing conditions, fun facts, wildlife value, planting guide, and detail photos.

**Browse all plants:** [ailula.top/Native-Plants-Card/plants/](https://ailula.top/Native-Plants-Card/plants/)

## Features

- Interactive card-flip design with plant info on the back
- Search and filter all 36 plants
- Save your favorite plants to "My Plants" collection
- Detail photos from iNaturalist research-grade observations
- Seasonal planting tips based on current month
